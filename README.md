Project Overview

This project analyzes Airbnb listing data to understand patterns in pricing, availability, listing distribution, and neighborhood demand. An interactive Power BI dashboard was created to explore market trends and help identify factors influencing listing performance.

Objectives
   - Analyze Airbnb listings and pricing patterns
   - Identify neighborhoods with the highest number of listings
   - Understand availability trends and host activity
   - Provide insights into market demand and property distribution

Tools & Technologies

- Power BI – Dashboard and data visualization
- Excel / CSV – Data cleaning and preparation
- DAX – Calculated measures and metrics

Key Dashboard Insights, 
   Some key insights from the analysis include:

- Certain neighborhoods have significantly higher listing density, indicating stronger tourism demand.

- Entire homes/apartments dominate the listing distribution, followed by private rooms.

- Pricing varies greatly across neighborhoods, suggesting location strongly influences listing value.

- Listings with higher reviews often show better availability patterns and stronger demand.


Dashboard Features

- Listing distribution by neighborhood

- Price analysis across room types

- Availability trends

- Host activity insights

- Interactive filters for deeper exploration
